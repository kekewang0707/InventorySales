# v0.1.0 — 产品库 + 客户管理

## 目标
完成基础数据管理：公司产品信息库 + 客户信息库。

---

## 1) 数据模型

### Product（产品）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer PK | 自增主键 |
| name | String(200) | 产品名称 |
| model | String(200) | 规格型号 |
| default_price | Decimal(10,2) | 默认单价（可空） |
| remark | Text | 备注 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### Customer（客户）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer PK | 自增主键 |
| name | String(200) | 客户名称/公司全称 |
| contact_person | String(100) | 联系人 |
| phone | String(50) | 联系电话 |
| address | Text | 地址 |
| remark | Text | 备注 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

---

## 2) API 端点

- `GET /api/products` — 分页列表（支持名称/型号搜索）
- `POST /api/products` — 新增产品
- `PUT /api/products/{id}` — 编辑产品
- `DELETE /api/products/{id}` — 删除产品
- `GET /api/customers` — 分页列表（支持名称/电话搜索）
- `POST /api/customers` — 新增客户
- `PUT /api/customers/{id}` — 编辑客户
- `DELETE /api/customers/{id}` — 删除客户

---

## 3) Vue 前端页面

- **左侧菜单**: 产品管理、客户管理
- **产品列表页**: El-Table 展示，顶部搜索框 + 新增按钮，每行编辑/删除操作
- **产品表单**: Dialog 弹窗表单，字段校验（名称必填）
- **客户列表页**: 同产品页样式，字段不同
- **客户表单**: Dialog 弹窗表单，字段校验（名称、电话必填）

---

## 4) PyQt6 桌面壳（共享基础）

- `desktop/app.py`: 启动 QApplication，加载 QWebEngineView 指向本地 FastAPI 服务
- `desktop/server_manager.py`: 子进程启动 uvicorn；窗口关闭时优雅终止

---

## 5) 测试

- 后端: 每个 API 端点 1 个 pytest 测试（CRUD 全流程 + 搜索过滤）
- 前端: 手动验证
