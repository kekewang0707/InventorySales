# v1.0.0 — AI 快捷操作

## 目标
用户通过自然语言（文本输入或语音转文字）描述需求，AI 理解语义并自动执行操作或给出应答。

---

## 1) 架构

```
用户输入文字
  → POST /api/ai/command { text: "..." }
  → ai_service.py:
      1. 调用 LLM API（OpenAI Function Calling）
      2. LLM 返回结构化操作指令
      3. 执行指令（调用已有 API/服务）
      4. 返回结果 + 自然语言回复
  → 前端展示
```

---

## 2) LLM Function Calling 工具

| 函数 | 说明 | 安全等级 |
|------|------|---------|
| search_product(query) | 搜索产品 | 查询 |
| search_customer(query) | 搜索客户 | 查询 |
| list_recent_delivery_notes(limit) | 最近送货单 | 查询 |
| create_delivery_note(params) | 创建送货单 | 需确认 |
| get_statement(customer, start, end) | 查询对账单 | 查询 |
| print_delivery_note(id) | 打印送货单 | 需确认 |

---

## 3) API 端点

- `POST /api/ai/command`
  - Request: `{ "text": "今天有没有送货单" }`
  - Response: `{ "action": "queried", "reply": "今天有3张送货单..." }`
- `POST /api/ai/confirm`
  - 用户确认 AI 要执行的写操作

---

## 4) 安全设计

- 查询操作直接执行
- 写入/删除操作需二次确认：前端弹确认框
- 所有操作记录 audit log

---

## 5) Vue 前端页面

- **AI 指令面板**: 浮窗或侧边面板，任意页面可呼出（Ctrl+K）
- 输入框 → 回车发送 → 展示结果
- 确认对话框（写操作）

---

## 6) 备选方案

- 离线模式: 关键词 + 规则匹配的命令解析（无 API 成本，功能降级）
