# v0.3.0 — 客户对账单

## 目标
统计指定时间范围内某个/所有客户的送货单汇总，生成对账单 PDF。

---

## 1) API 端点

- `GET /api/statements?customer_id=&start_date=&end_date=` — 返回对账单数据
  - 响应: 客户信息、送货单列表（编号、日期、金额）、合计金额
- `POST /api/statements/print` — 生成对账单 PDF

---

## 2) PDF 对账单

**模板内容**:
- 抬头: "客户对账单"
- 客户名称、对账期间
- 表格: 日期 | 送货单号 | 金额 | 备注
- 底部: 合计金额（大写 + 小写）

---

## 3) Vue 前端页面

- **对账单查询页**:
  - 客户选择器（支持"全部客户"）
  - 日期范围选择器（默认本月）
  - "查询" 按钮 → 显示统计表格
  - "打印对账单" 按钮 → 生成 PDF
- **导航**: 左侧菜单新增 "客户对账单"

---

## 4) 核心变更

- 无需新增数据模型，完全基于 DeliveryNote 查询聚合
- `pdf_service.py` 增加对账单 PDF 生成函数
- 新增 `routers/statements.py`
