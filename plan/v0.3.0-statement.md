# v0.3.0 — 客户对账单

## 目标

统计指定时间范围内某个/所有客户的送货单汇总，生成对账单 Excel。

---

## 1) API 端点

- `POST /api/statements` — 返回对账单数据
    - 响应: 客户信息、送货单列表（编号、日期、金额）、合计金额
- `POST /api/statements/export` — 生成对账单 Excel

---

## 2) Excel 对账单

**模板内容**:

- 抬头: "客户对账单"
- 客户名称、对账期间
- 表格: 日期 | 送货单号 | 明细【商品名称 * 数量】 | 金额 | 备注
- 底部: 合计金额（小写）

---

## 3) Vue 前端页面

- **对账单查询页**:
    - 客户选择器（支持"全部客户"）
    - 日期范围选择器（默认本月）
    - "查询" 按钮 → 显示统计表格
    - "导出对账单" 按钮 → 生成 excel，保存到本地，文件名格式：`客户名称_YYYY-MM-dd-YYYY-MM-dd.xlsx`
- **导航**: 左侧菜单新增 "客户对账单"
- 查询与导出使用相同请求体，避免筛选条件不同步。
- 导出时直接使用 axios 指定 responseType: 'blob'，从响应头获取文件名或按规则构造。

---

## 4) 核心变更

- 无需新增数据模型，完全基于 DeliveryNote 查询聚合
- `excel_service.py` 增加对账单 Excel 生成函数
- 新增 `routers/statements.py`
